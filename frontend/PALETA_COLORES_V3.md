# 🎨 PALETA DE COLORES VISIONQA V3 - DISEÑO CALMADO Y PROFESIONAL

## 🧘 Filosofía: Reducir Estrés Visual

**PROBLEMA ANTERIOR:** Exceso de rojo genera ansiedad y estrés constante  
**SOLUCIÓN:** Rojo SOLO para alertas críticas reales. Colores principales calmados y profesionales.

---

## 🎯 COLORES PRINCIPALES (Orden de Prioridad)

### 1. 🔵 AZUL SLATE - Color de Marca Principal
- **Color:** `#3b82f6` (Blue-500) / `#1e40af` (Blue-800 oscuro)
- **Uso:** Headers, tabs activos, botones principales
- **Psicología:** Calma, profesionalismo, confianza, estabilidad
- **Ejemplos:**
  - Header principal de la app
  - Tabs activos en navegación
  - Botones de acción principal (INICIAR, CONTINUAR, GUARDAR)
  - Enlaces y referencias

### 2. 🟢 VERDE ESMERALDA - Confirmación y Éxito
- **Color:** `#10b981` (Emerald-500)
- **Uso:** Estados positivos, aprobaciones, confirmaciones
- **Psicología:** Seguridad, éxito, todo está bien
- **Ejemplos:**
  - Lotes APROBADOS
  - Semáforo verde (calidad óptima)
  - Estados "OPERATIVO", "ACTIVO"
  - Botones de confirmación positiva

### 3. 💙 CYAN/TEAL - Información Neutral
- **Color:** `#06b6d4` (Cyan-500) / `#14b8a6` (Teal-500)
- **Uso:** Datos informativos, estadísticas, métricas
- **Psicología:** Claridad, transparencia, modernidad
- **Ejemplos:**
  - KPIs y métricas
  - Contadores de unidades
  - Información técnica
  - Badges informativos

### 4. 🟣 ÍNDIGO/MORADO - Funciones Especiales
- **Color:** `#6366f1` (Indigo-500) / `#8b5cf6` (Violet-500)
- **Uso:** Administración, configuración, supervisor
- **Psicología:** Especialización, control avanzado
- **Ejemplos:**
  - Módulo de Administración
  - Funciones de supervisor
  - Configuración avanzada

### 5. 🟡 ÁMBAR - Precaución (No Alarma)
- **Color:** `#f59e0b` (Amber-500)
- **Uso:** Advertencias que requieren atención, no urgencia
- **Psicología:** Atención sin estrés, precaución razonable
- **Ejemplos:**
  - Lente necesita limpieza
  - Calibración próxima
  - Lote en zona de precaución (1.0-1.5% defectos)

### 6. 🔴 ROJO - **SOLO CRÍTICOS REALES**
- **Color:** `#ef4444` (Red-500)
- **Uso:** **SOLO emergencias y errores críticos que requieren acción inmediata**
- **Psicología:** PELIGRO, DETENER, ACCIÓN URGENTE
- **Ejemplos (LIMITADOS):**
  - Lote RECHAZADO (>2.5% defectos, superó AQL)
  - Parada de emergencia
  - Error de sistema crítico
  - Alerta de seguridad

## 🧘 PRINCIPIOS DE DISEÑO CALMADO

### 1. Jerarquía de Urgencia
```
🔵 Azul (Neutro) → 💙 Cyan (Info) → 🟢 Verde (Positivo) 
                ↓
🟡 Ámbar (Precaución) → 🔴 Rojo (CRÍTICO)
```

### 2. Regla del Rojo
- ❌ NO usar rojo en: headers, sidebar, tabs, badges de módulo
- ❌ NO usar rojo para "importancia" o "destacar"
- ✅ SOLO usar rojo cuando: hay un problema crítico REAL que requiere acción inmediata

### 3. Equilibrio Visual
- **70%** Azules y grises (calma)
- **20%** Verdes y cyans (positivo/neutro)
- **9%** Ámbar (precaución)
- **1%** Rojo (solo críticos)

### 4. Colores Calmantes
- Gradientes suaves en lugar de bloques sólidos
- Sombras sutiles con blur
- Bordes en tonos azul/slate en lugar de rojo
- Animaciones suaves (no pulse agresivo constante)

## 🎨 CÓDIGO DE REFERENCIA

```tsx
// NUEVA paleta principal
const colors = {
  // Primarios (usar más)
  primary: "#3b82f6",      // Azul - Marca principal
  success: "#10b981",      // Verde - Confirmaciones
  info: "#06b6d4",         // Cyan - Información
  admin: "#6366f1",        // Índigo - Admin
  
  // Secundarios (usar moderadamente)
  warning: "#f59e0b",      // Ámbar - Precaución
  danger: "#ef4444",       // Rojo - SOLO críticos
  
  // Fondos (sin rojo)
  bg: {
    base: "#0a0a0a",
    card: "#1a1a1a",
    element: "#2a2a2a",
    hover: "#3a3a3a"
  }
};

// Header AZUL (no rojo)
<div className="
  backdrop-blur-xl 
  bg-[rgba(30,64,175,0.8)]
  border-b-2 border-[rgba(59,130,246,0.3)]
  shadow-[0px_0px_30px_0px_rgba(59,130,246,0.2)]
">
  {/* Contenido */}
</div>

// Tab activo AZUL (no rojo)
<button className={`
  ${isActive 
    ? "bg-[#3b82f6] text-white" 
    : "bg-[#2a2a2a] text-[#71717a]"
  }
`}>
  {tab.label}
</button>

// Semáforo INTELIGENTE (rojo solo cuando es crítico)
<div className={`
  ${tasa < 1.0 ? "bg-[#10b981]" :      // Verde
   tasa < 1.5 ? "bg-[#06b6d4]" :       // Cyan
   tasa < 2.5 ? "bg-[#f59e0b]" :       // Ámbar
   "bg-[#ef4444]"}                      // Rojo CRÍTICO
`}>
```
